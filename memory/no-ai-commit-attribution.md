When committing, do NOT add `Co-Authored-By: Claude`, "Generated with Claude Code", or any AI-attribution trailer/footer. Write commits as plain human edits.

**Why:** The user wants the project history to read as authored by them.

**How to apply:** Use a normal commit message with no AI trailer. This applies to every commit in this project.
